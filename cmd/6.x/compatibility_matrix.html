Sencha Cmd 兼容性列表

推荐的 Sencha Cmd 版本

推荐使用最新版的 Sencha Cmd，因为它包含了最近的 bug 修复. 我们力争每次发布都继续支持以下版本的框架:

  • Ext JS 4.1.1a+
  • Sencha Touch 2.1.0+

注意: Cmd 5.x 不再支持 ST Build. ST Build 在 Cmd 4.x 中仍然可用, 但是 Cmd 5+ 需要 Cordova或 PhoneGap 来原生打包.

当前版本

如果你已经装了 Sencha Cmd 3.1 或更高版本, 你可以用以下命令检查最近稳定版本:

sencha upgrade --check

同样, 你也可以检查最近的 beta 版本:

sencha upgrade --beta --check

如果要下载最近版本，请移除“–-check”然后执行. 下载完毕后会自动启动安装程序.

注意: 最近的稳定版可能比最近 beta 版要新，反之亦然.
